# Lode Runner

A browser-based Lode Runner clone written entirely by Claude (Anthropic's AI assistant).

**[Play it here](https://tmikov.github.io/lode-runner/)**

## Controls

| Action | Keys |
|--------|------|
| Move | Arrow keys or WASD |
| Dig left | Z or Q |
| Dig right | X or E |
| Pause | P or Escape |
| Restart | R |
